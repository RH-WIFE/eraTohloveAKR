;安定口上について
;--------------------------------------------
■基本的には穏やかで物腰柔らかく、しかし戦闘ともなると一変して首落とそうとしてくる安定くんを目指して作成しています。
　ただし戦闘をするシステムはないため、淫乱系に堕ちた場合にオラオラするようになりました。

■「気丈」に振る舞い「反抗的」で、「一線越えない」「ツンデレ」。
　普段は強気ですが、ストレスが溜まると途端に弱音を吐き始めます。
　現状、陥落した状態でストレスが溜まっても同じように弱音を吐き始めるため、改変が必要かもしれません…今後の課題。

■この口上自体、現状、ほぼ未完成です。
　ストーリー系がちょこっとと、労役をさせるときに喋るくらい。
　喘ぎ声は入っていないため、「汎用喘ぎを使用する」にチェックを入れていれば、汎用喘ぎが適用されます。

■加州清光、和泉守兼定、堀川国広に少しだけ対応しています。
　上記３人が主人（もしくは助手）の場合に、特殊な会話が発生するようになります。
　ただし基本的には「審神者」が相手であることを前提として作っているため、場合によっては違和感があるかもしれません。
　陸奥守は一瞬対応させようとして、土佐弁が書けずに諦めました。誰か書いてください…。
　上記の口上を書かれる方で、齟齬が生じる等の場合にはお好きに書き直してやってください。

■安定から主人（審神者限定）への、呼び名を変更することができます。
　「初めて安定を調教するとき」「いずれかの素質に陥落後、安定と[会話]するとき」に、呼び名変更が可能です。
　任意の文字列を入力することもできますので、お好きな呼ばれ方で楽しんでみてください。

■さらに詳しいコンセプトや設定は、ページ下部にあります。
　完全にネタバレなので、スクロールには注意してください。
　避けられる地雷は避けて通るのが、賢い審神者のマナーだよ☆

;基本スペック
;--------------------------------------------
■体力面
　体力1900、気力1300。普通の打刀。
　体力は加州より若干高く、気力は加州より若干低い。
■技術面
　特筆事項無し。普通の打刀。

■性格
　一見従順で大人しそうに見えるが、実はかなり我が強く、【反抗的】で【気丈】な性格。
　沖田くんの事は敬愛しているが、その代わり主のことは信用していない。
　１人に固執するタイプであるのか、堕ちる前はツンツン、
　ただし一度堕ちてさえしまえば真っ逆さまの【ツンデレ】持ち。
　性的な行為に関しては若干【保守的】な面があり、、かつ最後の最後で【一線越えない】。
　扱いづらい、という特性が表れている。


;--------------------------------------------
;陥落までの流れとその後
;--------------------------------------------
■初期
　戦うために呼ばれた、と思ったら愛玩用の玩具になれと言われて戸惑う安定。
　そんなことに使われたくないと、主人に対して反抗的な態度を取りまくる。

■恋慕
　何故か優しい主にほだされかけ。…僕を、愛してくれるのかい？
■親愛
　愛してもらえてる自覚が出てきた安定。…沖田くん、僕、好きな人ができたよ。

■淫乱
　気持ちいいのに慣れてきた。…これで、相手が好みの奴だったら…なんでもない。
■娼婦
　肉欲に溺れた子猫ちゃん。…合わさるって、いいよね。
　　→テンションが上がると戦闘モードになります。

■服従
　自分は所詮刀だから、逆らう気力が失せてきた。…目をかけられてる分は、頑張るよ。
■隷属
　性奴隷である自覚が出てきた。…あなたが新しい、僕の大切な主です。

■崩壊
　見えちゃいけないモノが見えるようになった。
　…主？　加州清光？　和泉守兼定？　みんな沖田くんに見えるよ。
